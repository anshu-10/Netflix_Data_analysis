# Netflix Data Analysis

## Overview

This project focuses on performing Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset using Python.
The goal is to clean the data, analyze content trends, and visualize insights such as release patterns, genres, and content distribution.

---

## Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

---

## Dataset

* Netflix Movies & TV Shows Dataset (CSV format)
* Contains information such as title, type, release year, country, genre, and ratings.

---

## Key Analysis Performed

* Data cleaning and handling missing values
* Content growth analysis by year
* Genre and category distribution
* Movies vs TV Shows comparison
* Visualization of trends using graphs

## Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/anshu-10/Netflix_Data_analysis.git
```

### 2. Navigate to the project folder

```
cd Netflix_Data_analysis
```

### 3. Install required libraries

```
pip install pandas numpy matplotlib
```

---

## Usage

Open the Jupyter Notebook and run all cells step by step:

```
jupyter notebook
```

## Results and Insights

* Significant growth in Netflix content after 2016.
* Movies dominate the platform compared to TV Shows.
* Certain genres show higher yearly growth trends.
